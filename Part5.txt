📊 ANALYTICS DATA POINTS TO COMPUTE
═══════════════════════════════════════════════
All computed from existing module tables (no raw data duplication).

A. PROJECT ANALYTICS
   - Total projects by status
     (Draft/Active/OnHold/Completed/Cancelled)
   - Project completion rate (%)
   - Average project duration (days)
   - Projects at risk (past EndDate, not Completed)
   - Projects created per month (last 12 months)
   - Project progress % per project
     (completed tasks / total tasks)
   - Milestone completion rate per project

B. TASK ANALYTICS
   - Total tasks by status per project
   - Task completion rate (%) per project / per user
   - Average task resolution time (days)
   - Overdue tasks count (past DueDate, not Done)
   - Tasks created vs completed per week
   - Task distribution by priority
   - Top performers (users with most tasks completed)

C. TEAM PRODUCTIVITY ANALYTICS
   - Tasks completed per team member (last 30 days)
   - Average task completion time per user
   - Team member workload (assigned open tasks count)
   - Most active contributors (by task updates + doc uploads)
   - User activity heatmap data (actions per day of week)

D. DOCUMENT ANALYTICS
   - Total documents uploaded per project
   - Documents by file type breakdown
   - Upload activity over time (per month)
   - Most downloaded/viewed documents (top 10)
   - Storage used per organization (MB)
   - Document version update frequency

E. COLLABORATION ANALYTICS
   - Total messages sent per project channel
   - Most active channels
   - Announcement read rates (read / total members %)
   - Notification response rate
   - Active users per day/week

F. INNOVATION OVERVIEW (Executive Dashboard)
   - Total active projects
   - Total team members
   - Overall task completion rate
   - Total documents in system
   - Projects completed this quarter
   - R&D velocity (tasks completed per week trend)
   - Innovation score (composite KPI –
     weighted average of completion rate,
     on-time delivery, team activity)

6. INNOVATION SCORE FORMULA
   Compute as weighted composite (0–100 scale):

   InnovationScore = (
     (ProjectCompletionRate  × 0.25) +
     (OnTimeDeliveryRate     × 0.25) +
     (TaskCompletionRate     × 0.20) +
     (TeamActivityScore      × 0.15) +
     (DocumentationRate      × 0.10) +
     (CollaborationScore     × 0.05)
   ) × 100

   Where:
   - ProjectCompletionRate  = completed / total projects
   - OnTimeDeliveryRate     = projects completed before
                              EndDate / total completed
   - TaskCompletionRate     = done tasks / total tasks
   - TeamActivityScore      = active users last 30d /
                              total users
   - DocumentationRate      = projects with ≥1 document
                              / total projects
   - CollaborationScore     = projects with ≥1 channel
                              message / total projects

   Return score + breakdown per component in
   InnovationScoreDto.cs
